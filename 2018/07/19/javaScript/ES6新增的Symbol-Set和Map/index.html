ES6新增的Symbol,Set和Map

发表于 2018-07-19 | 分类于 javaScript

Symbol

  • 每个Symbol返回的值都是唯一的。
  • 一个Symbol的值可以做为对象的属性。
  • Symbol的参数可选，一般用于调式。

如何创建一个Symbol？

变量s就是一个Symbol.

如果Symbol的参数是一个对象，就会调用对象的toString方法，转为字符串，然后才生成一个Symblo值。

因为每个Symbol返回的值都是唯一的，因此相同函数的Symbol的返回值是不想等的。

作用

Symbol用作于隐藏，私有属性，在ES6之前，是无法表示一个隐藏属性的。因为ES6之前对象的属性都是字符串。

通过Symbol定义的属性，其他人是无法访问的，只有自己可以访问修改。

Set

  • set是一种特殊的对象
  • 允许存储任何对象的唯一值，重复的会被过滤，无论原始值还是对象引用。

常用方法：

  • add（vlue） 尾部添加一个元素
  • delete（value）移除set中与这个对象相等的值
  • clear 移除set对象内所有元素
  • has 判断是否存在，返回true或false
  • keys 返回对象中所有的key值
  • values 返回对象中所有的value值
  • entries 返回对象中所有的key，value
