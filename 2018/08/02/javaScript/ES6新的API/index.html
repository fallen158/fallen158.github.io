Object新增的API

  1. 属性简洁表示法

除了属性简写，方法也可以简写

  1. Object.is()

ES5 比较两个值是否相等，只有两个运算符：相等运算符（==）和严格相等运算符（===）。它们都有缺点，前者会自动转换数据类型，后者的NaN不等于自身，以及+0等于-0。

  1. Object.assign

基本用于对象的合并，将源对象复制到目标对象

如果源对象和目标对象又同名属性，则后面的属性会覆盖前面的属性。

需要注意的是Object.assign 是浅拷贝，而不是深拷贝。也就是说，如果源对象某个属性的值是对象，那么目标对象拷贝得到的是这个对象的引用

浅拷贝: 把所有东西拷贝一份。

深拷贝：除啦内存地址以外，所有的东西都拷贝一份，少一个都不算。

常见用途

为对象添加属性

为对象添加方法

克隆这个对象继承的值写法

合并多个对象

Number新增的API

1.isInteger 判断一个数字是否为整数

2.isFinite 判断是否为有限的数字，返回true或false

3.isNaN

判断是否为NaN

ES5的时候是根据不等于自己来判断NaN的

以及window.isNaN

但他有一个弊端，如果是传入字符串的话，它也会返回true，它会帮你把传入的字符串parseFloat一下。

ES6 最新的写法 Number.isNaN 它纠正啦上面的错误 。

ES6还把全局方法parseInt(),和parseFloat（）移值到Number对象上面。

Math新增的API

1.Math.trunc 去除一个数的小数部分，返回整数部分

2.Math.sign 判断一个数字是正数，还是负数，0

参数为正数，返回 + 1；

参数为负数，返回 - 1；

参数为 0，返回0；

参数为 - 0，返回 - 0;

其他值，返回NaN。
